# vagrantbox

base vagrant box to use for rails course

## Installation

```bash
git clone https://github.com/dojorailsMAY/vagrantbox.git
cd vagrantbox
vagrant up
# ... Wait ...
vagrant ssh
# Ubuntu welcome screen
rails -v
# 4.2.7
```

## Usage

To enter machine:
```bash
vagrant ssh
```

To exit machine:
```bash
exit
```

To shutdown machine:
```bash
vagrant halt
```

To power on machine:
```bash
vagrant up
```

To restart machine:
```bash
vagrant reload
```
